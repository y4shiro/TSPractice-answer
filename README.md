## 実行方法

TS ファイルを `src/` 配下に設置し、下記コマンドで TS を直接実行

```
$ npm run dev <ファイル名>
```

TS ファイルの変更を監視して継続的に実行したい場合は下記コマンド

```
$ npm run dev:watch <ファイル名>
```
